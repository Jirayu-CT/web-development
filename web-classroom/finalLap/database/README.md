# ตาราง `books`

ตาราง `books` ถูกใช้เพื่อเก็บข้อมูลเกี่ยวกับหนังสือในฐานข้อมูล โดยมีคอลัมน์ต่างๆ ดังนี้:

- **id**: รหัสหนังสือ เป็นคีย์หลักและเพิ่มค่าอัตโนมัติ
- **title**: ชื่อหนังสือ
- **author**: ชื่อผู้เขียน
- **genre**: ประเภทของหนังสือ
- **published_year**: ปีที่ตีพิมพ์
- **isbn**: รหัส ISBN ของหนังสือ
- **pages**: จำนวนหน้าของหนังสือ
- **publisher**: สำนักพิมพ์
- **language**: ภาษาของหนังสือ
- **description**: คำอธิบายเกี่ยวกับหนังสือ
